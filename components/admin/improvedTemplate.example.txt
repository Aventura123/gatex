Placeholders disponíveis:
- {{title}} - Título da vaga
- {{companyName}} - Nome da empresa 
- {{id}} - ID da vaga
- {{jobUrl}} - URL completa da vaga (https://gate33.io/jobs/ID)
- {{mediaUrl}} - URL da imagem do post

Dicas para um bom post:
1. Use emojis para destacar informações importantes
2. Inclua sempre a URL da vaga {{jobUrl}}
3. Use hashtags relevantes
4. Mantenha o texto conciso
5. Use uma imagem de qualidade para aumentar o engajamento
